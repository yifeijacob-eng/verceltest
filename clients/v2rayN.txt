# ============================================
# v2rayN / v2rayNG 配置
# ============================================

复制以下链接到 v2rayN / v2rayNG:

vless://12345678-1234-1234-1234-123456789abc@your-app.vercel.app:443?encryption=none&security=tls&type=ws&host=your-app.vercel.app&path=%2Fapi%2Fvless#vercel-vless

# ============================================
# 参数说明
# ============================================
UUID:     12345678-1234-1234-1234-123456789abc  (替换为你的 UUID)
服务器:   your-app.vercel.app                    (替换为你的 Vercel 域名)
端口:     443
加密:     none
传输:     ws (WebSocket)
TLS:      开启
Path:     /api/vless
Host:     your-app.vercel.app
